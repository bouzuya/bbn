# bbn

A command-line tool for [blog.bouzuya.net](http://blog.bouzuya.net).

See: [bouzuya/blog.bouzuya.net][]

## Installation

    $ npm install -g bouzuya/bbn

## Usage

    $ cat ~/.bbn.json
    {
      "directory": "/home/bouzuya/blog.bouzuya.net"
    }

    $ bbn new
    create a new post "2014-06-02-diary.markdown"
